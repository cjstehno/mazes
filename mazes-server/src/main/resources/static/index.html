About

This Spring-Boot application was created after working through the examples in the "Mazes For Programmers" by Jamis Buck. While reading the book, I ported the code from Ruby over to Groovy, and when I finished I thought it would be interesting to create a little maze server using the techniques learned from the book.

This application (and the ported code from the book) are available on my mazes GitHub project.
